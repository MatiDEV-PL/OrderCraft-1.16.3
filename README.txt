To run this mod in dev version you need to insall:
-Java SE Development Kit 8 link:https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html
-IntelliJ IDEA link:https://www.jetbrains.com/idea/download/#section=windows
After extracting OrderCraft-1.16.2-master in IntelliJ IDEA you open existing project (in our case it would be OrderCraft-1.16.2-master folder), it should load, then in your right top screen you will see star icon. Remember to star it as runClient. And now you have full acces to the lastes dev version of OrderCraft.
